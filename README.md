使用过各种各样记笔记的姿势，最后发现还是在 IDE 里最方便。

设置个别名，方便的很。
```bash
vim ~/.zsh_rc

# add alias
alias vv="/Applications/Visual\ Studio\ Code.app/Contents/Resources/app/bin/code"
alias note="cd /Users/go_songs/study/notes; vv ."
```
